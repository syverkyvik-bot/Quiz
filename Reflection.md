# Reflection Notes - Quiz Project

I developed this quiz application independently, working with C++ to combine file I/O, object-oriented programming, and GUI development using the AnimationWindow library. 
One decision giving the GUI class direct access to the questions object rather than passing individual data, which simplified the architecture and improved code clarity. 
Another decision was to use plain text instead of TextBox widgets for displaying questions because it provided better control over text size and visual presentation. 
Future improvements could include timers, difficulty levels, randomized question numbers and input new questions throught the GUI.
- File I/O operations and string parsing
- Event-driven programming with callbacks
- Working with external libraries and build systems
- GUI programming with vector-based UI widgets
